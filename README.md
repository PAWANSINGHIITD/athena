# athena assignment
This project delivers an interactive, educational application that explores how different world cultures construct and represent numbers. The app integrates mathematics, linguistics, and cultural reasoning to help learners understand numeral systems through structured explanations and hands-on exploration.

The solution includes:

A Numeral System Library featuring Roman, Mayan, Chinese, Babylonian, Yoruba, Inuktitut, and other global systems. Each entry explains the base structure, morphology, recursive logic, and cultural background behind the numerals.

A Bidirectional Converter Engine capable of translating Arabic numbers into cultural numeral forms and converting cultural representations back to Arabic, with step-by-step explanations for conceptual clarity.

An Olympiad-Style Practice Zone offering pattern identification tasks, number-construction challenges, and Linguistics Olympiad–style puzzles designed to build analytical and cultural insight.

A Fully Functional Prototype built using Python (Flask), featuring modular architecture so additional numeral systems or puzzle types can be added easily.

Overall, the application bridges mathematical structure and linguistic patterning while offering a culturally enriched learning experience.
